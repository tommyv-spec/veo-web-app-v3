How It Works

1
Generate a token above This links your worker to your account. The setup command auto-includes it.
2
Run the setup command Installs Python packages, Playwright browser, and downloads the worker â€” all automatic. You'll pick which Chrome profile to use.
3
Select "Flow" as your method In the main app's Settings card, set Method to Flow. Then submit your job as normal.
4
Worker processes your job Your worker picks up the job, opens Google Flow, generates clips, and uploads the results back to the web app.
Your Chrome can stay open normally. The worker uses a copy of your Chrome profile, not the original.

After Setup

Your worker is installed at:

Windows: C:\Users\YourName\veo-worker\
Mac/Linux: ~/veo-worker/
Start worker (Windows):
cd $HOME\veo-worker; python flow_worker.py --single
Start worker (Mac/Linux):
cd ~/veo-worker && python3 flow_worker.py --single
Switch Chrome profile:
cd ~/veo-worker && python setup_worker.py --reselect-profile
Update worker:
cd ~/veo-worker && python setup_worker.py --update
